SIT223-SIT753 - Professional Practice in IT

This unit introduces students to IT workflows, agile project management,
DevOps pipelines, and version control, exploring how these tools integrate
within modern organisations. It also emphasises the professional behaviours,
collaborative teamwork, and ethical responsibilities essential for working effectively in the IT industry.
